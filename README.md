KTC – Transportation Management System 🚚

🌟 Overview
KTC (Kaushik Transport Company) is a full-stack Transportation Management System that digitalizes logistics operations including docket creation, billing, freight calculation, and shipment tracking.

Built for real industry workflows, this project serves as both a functional business solution and a comprehensive training reference for full-stack development.

🎯 Key Objectives
✅ Eliminate manual paperwork in transport operations
✅ Centralize shipment, billing, and customer data
✅ Real-world MERN stack implementation
✅ Training project for freshers & students

🛠️ Tech Stack
Frontend	Backend	Database
React.js	Node.js + Express.js	SQL (MySQL/PostgreSQL)
HTML5/CSS3/JS	RESTful APIs	Relational Schema
Component Architecture	JWT Ready	Optimized for Logistics

🚀 Quick Start (5 Minutes)
1. Clone & Backend
git clone https://github.com/Sahil-Kaushik1507/KTC.git
cd Backend-KTC
npm install
npm start
- API runs on http://localhost:5000

2. Frontend
cd ../Frontend-KTC
npm install
npm start
- App runs on http://localhost:3000

3. Database
-- Import Database-KTC/schema.sql
-- Update Backend-KTC/.env with DB credentials
✅ Done! Open http://localhost:3000
