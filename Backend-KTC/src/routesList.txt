main app : http://localhost:8001/api/v1/


Branch:
    http://localhost:8001/api/v1/branch

        /new --> Add New Branch (Post req)
        /addManager --> Add Manager to the existing Branch (Post Req)
        /viewAllBranches --> View All saved Branches(get req)
        /viewBranch/:branch_id --> Get particular branch details by ID (get with parms)


Authentication:
     http://localhost:8001/api/v1/auth

        /register --> Add new User (Post Req)
        /login --> Login (Post Req)
        /updateUserDetails --> Updateuser details ["phone_no", "email", "branch_id", "salary", "role"] (Patch Req)
        /changePassword --> change password (Patch req)
        /:user_id/deactivateUser --> Deactivate User (patch with Parms)

Parties:
    http://localhost:8001/api/v1/parties
        /new --> Add new Party (Postr Req)
        /get/:party_code Get Party Details (get with Params)
        ./get get all parties details (get req)

sequence
    http://localhost:8001/api/v1/sequence
        /addNewSequence --> Add new Sequence (post req)
        getNextNumber --> get sequence next number (patch req)
